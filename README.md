# MiseEnPlace
Social Media Home Chef Cooking Web Application.

This is the development of a new web application to allow users to do the following:
1. Create a personal profile
2. Build their "Cabinet/pantry" with an inventory of the users ingredients and equipment
3. View all recipes that they can make based on the users current ingredients and equipment 
4. Create a post of their own recipes
5. Follow other users to view their recipes
6. Develop a following
7. Save a cookbook "Favorited" items or User's own created recipes
8. Comment on recipes and add likes
9. Users can be paid out from advertisements
10. Recipes can embedd Youtube videos
